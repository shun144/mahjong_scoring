# 麻雀の点数計算ガイド｜符・翻から点数を出す流れ

点数計算は、料理のレシピと同じで「作る順番」が決まっています。順番さえ覚えれば、どんな手でも同じやり方で点数を出せます。このページで全体の流れをつかんで、細かい部分は各記事で深掘りしてください。

> 「符」「翻（ハン）」「点数」の関係をつかんだら、[麻雀点数トレーニング](/)でガンガン練習してみてください。

## 点数が決まる3ステップ

![符→翻→点数の3ステップ図](/articles/guide/three-steps.png)

点数は、次の3ステップで必ず決まります。

1. **符を数える**（あがりの「形」を点にする）
2. **翻を数える**（役とドラを合計する）
3. **符と翻を点数に変える**（計算して点数を出す）

この順番はどんな手でも変わりません。まずはこの3つがあることだけ覚えてください。

### このページでずっと使う例

説明がバラバラにならないよう、次の1つの手をゴールまで一緒に追いかけます。

> **例：門前（鳴いていない）でツモあがり。リーチもかけた。5萬が3枚そろった暗刻があり、タンヤオも成立。待ちは両面待ち。**

この手を最後に点数まで出しきります。

## ステップ1：符を数える

点数計算で一番の難所がこの符です。ここさえクリアすれば、あとは一気に楽になります。

符とは、かんたんに言うと「あがりの形の複雑さ」を点にしたものです。難しい形であるほど符が高くなります。

### まず20符からスタート

どんな手も、最初に**20符**をもらえます。ここから、条件を満たすたびに符をたし算していきます。

たす条件は、大きく分けて4つです。

| たす場所 | 内容                                 | 加える符        |
| -------- | ------------------------------------ | --------------- |
| あがり方 | ツモ or 門前でロン                   | +2符または+10符 |
| 面子     | 刻子や槓子の種類による（下の表）     | +2〜+32符       |
| 雀頭     | 役牌のペア（白・發・中・場風・自風） | +2符            |
| 待ち     | カンチャン・ペンチャン・単騎         | +2符            |

### あがり方の符

ツモの場合は、どんな時でも+2符が付きます。

ロンの場合は、門前（鳴いていないこと）の時のみ+10符が付きます。
なので鳴いてしまった場合のロンは0符となります。

### 面子（刻子・槓子）の符

同じ牌をそろえた面子は、次のように符がつきます。

| 面子の種類        | 中張牌（2〜8） | 么九牌（1・9・字牌） |
| ----------------- | -------------- | -------------------- |
| 明刻（鳴いた3枚） | 2符            | 4符                  |
| 暗刻（自力の3枚） | 4符            | 8符                  |
| 明槓（鳴いた4枚） | 8符            | 16符                 |
| 暗槓（自力の4枚） | 16符           | 32符                 |

覚え方はシンプルで、**「隠しているほど（暗いほど）」「端っこの牌（么九）ほど」高くなる**、と考えると迷いません。

なお、順子（連番3枚）は0符です。同じ牌をそろえた面子だけに注目してください。

コツとしては**暗刻の2～8牌は4符**と**暗刻の1・9・字牌は8符**を基準にすると覚えやすいです。

### 待ちの種類

あと1枚を待つ形にも符がつきます。

| 待ちの種類         | どんな待ち                            | 加える符 |
| ------------------ | ------------------------------------- | -------- |
| 両面（リャンメン） | 例：34 で 2 か 5 を待つ               | 0符      |
| シャンポン         | ペア2組でどちらかが刻子になるのを待つ | 0符      |
| カンチャン         | 例：13 で間の 2 だけを待つ            | +2符     |
| ペンチャン         | 例：12 で 3 だけ／89 で 7 だけを待つ  | +2符     |
| 単騎（タンキ）     | 雀頭のもう1枚を待つ                   | +2符     |

### 例で符を数えてみる

![符の内訳の積み上げ図](/articles/guide/fu-breakdown.png)

さっきの例（門前ツモ・5萬の暗刻・両面待ち）で数えてみましょう。

- 基本：**20符**
- ツモであがった：**+2符**
- 5萬（中張牌）の暗刻：**+4符**
- 両面待ち：**+0符**
- 雀頭は普通の牌：**+0符**

合計は **20＋2＋4＝26符**。

最後に**10符の位で切り上げ**るので、26符 → **30符**になります。この「切り上げ」を忘れる人がとても多いので注意してください。

### 覚えておく2つの例外

- **平和ツモ**は計算せず **20符で固定**
- **七対子**は計算せず **25符で固定**

詳しくは「符計算のやり方」へ。

## ステップ2：翻を数える

翻は、成立した**役の翻数＋ドラの数**をぜんぶ足したものです。役が1つもないとあがれない、という点にも注意してください。

![主要な役の翻数の例](/articles/guide/han-table.png)
_リーチ1翻・タンヤオ1翻・ホンイツ2/3翻…など主要役を一覧表で_

さっきの例では、次の役がそろっています。

- リーチ：1翻
- 門前ツモ：1翻
- タンヤオ：1翻

合計で **3翻** です。（今回はドラなし）

なお、鳴くと翻が減る役（食い下がり）もあります。詳しくは「役一覧」へ。

## ステップ3：点数に変換する

いよいよ、符と翻を点数に変えます。使う式はこれ1本です。

```
基本点 = 符 × 2^(2 + 翻)
```

`2^(2+翻)` は「2 を（2＋翻数）回かける」という意味です。今回は3翻なので、`2 + 3 = 5`。**2を5回かけると 32** になります。

だから、さっきの例（30符・3翻）の基本点は、

```
基本点 = 30 × 32 = 960
```

この**基本点に、親か子か・ロンかツモかで倍率**をかけて、実際の点数にします。

![点数変換の早見イメージ](/articles/guide/score-convert.png)
_子ロン×4／親ロン×6／ツモ時の親子の支払い_

| あがり方 | かけ算         | 今回の点数（切り上げ後）       |
| -------- | -------------- | ------------------------------ |
| 子のロン | 基本点×4       | 3900点                         |
| 子のツモ | 親は×2／子は×1 | 親2000＋子1000ずつ＝**4000点** |

例は「子のツモ」なので、親から2000点・ほかの子2人から1000点ずつ、**合計4000点**が答えです。

このとき、支払いは**100点の位で切り上げ**ます（960×2＝1920 → 2000）。

細かい組み合わせは「点数早見表」へ。

## 満貫以上の区切り

翻がどんどん増えると点数は青天井にはならず、区切りのところで固定されます。

![満貫〜役満の点数表](/articles/guide/mangan-table.png)
_満貫・跳満・倍満・三倍満・役満の子/親点数_

| 区切り | 子のあがり | 親のあがり |
| ------ | ---------- | ---------- |
| 満貫   | 8000点     | 12000点    |
| 跳満   | 12000点    | 18000点    |
| 倍満   | 16000点    | 24000点    |
| 三倍満 | 24000点    | 36000点    |
| 役満   | 32000点    | 48000点    |

## もう1つの例：リーチ・ツモ・平和・ドラ1（子）

平和ツモは20符固定、というルールを使う例です。

- 符：**20符**（平和ツモは固定）
- 翻：**4翻**（リーチ＋ツモ＋平和＋ドラ1）
- 基本点：20 × 2^(2+4) ＝ 20 × 64 ＝ **1280**
- 子のツモなので、親は1280×2＝2600、子は1280→1300
- 答え：**親2600／子1300ずつ → 合計5200点**
  慣れるまでは[ツール](/)で答え合わせしながら、実際に手を動かすのが一番の近道です。

## つまずきやすいポイント

- **平和ツモを30符にしてしまう**（正しくは20符固定）
- **七対子を符計算してしまう**（25符で固定）
- **ツモの+2符を足し忘れる**
- **切り上げを忘れる**（符は10の位、点数は100の位で切り上げ）
  →「よくある計算ミス」でまとめて解説。
