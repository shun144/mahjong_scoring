# 麻雀記事 企画一覧

初心者に麻雀のルールを楽しく覚えてもらうための記事候補一覧。

## ルールの基本を掴む記事

[x] 1局の流れを最初から最後まで（配牌→ツモ→打牌→アガリ／流局）

1. ツモる（山から牌を引く）
2. 役がそろうか考える
3. 打牌（一枚捨てる）
4. アガる or 流局
   [] 最初に覚える役ベスト5〜7（リーチ・タンヤオ・ピンフ・役牌・ホンイツなど）
   [x] 役一覧
   [x] フリテンとは
   [] リーチの詳しいルール（1000点棒・一発・裏ドラ・手替え不可など）

## 初心者がとまどうところ

[] 親の決め方
[] 山は最初どこからとる
[] カンした時のめくりかた

## 実戦で覚えること

[] 鳴きの細かいルール（ポン・チー・カンの優先順位、鳴ける牌／鳴けない牌）
[] 牌効率のいろは（何を切ればいいか、両面優先など）
[] 危険牌のかんたんな見分け方（スジ・現物）
[] 麻雀のマナー・卓上のふるまい（先ヅモ・発声のタイミングなど）

## 点数まわり（計算しなくても分かる版）

[] 翻と符と点数の早見表
[] 満貫・跳満・倍満ってなに？

## 点数計算を深く知りたい人向け

[] 符計算のやり方（図解）
[] よくある計算ミス集
[] 満貫・跳満・倍満の境界（点数計算ガイドからの続き）

## 特殊ルール・応用

[] 途中流局のルール（九種九牌・四風連打・四家立直・四開槓など）
[] ダブロン・トリロンの扱い
[] 数え役満・切り上げ満貫などの特殊ルール
[] 手積みと全自動卓の違い
