# 麻雀 基本用語集｜牌・進行・役でよく出てくる言葉

麻雀でよく出てくる言葉を説明します。

> 役の説明はこの記事には含まれません。

## 牌に関する用語

| 用語                       | 説明                                                      | 画像 |
| -------------------------- | --------------------------------------------------------- | ---- |
| 牌（パイ）                 | 麻雀で使うカードのようなコマのこと。全部で136枚あります。 |      |
| 萬子（マンズ）             | 数字と「萬」の字が書かれた牌のグループのこと。            |      |
| 筒子（ピンズ）             | 丸い模様が描かれた牌のグループのこと。                    |      |
| 索子（ソウズ）             | 棒のような模様が描かれた牌のグループのこと。              |      |
| 字牌（ジハイ）             | 東・南・西・北や、白・發・中が書かれた牌のこと。          |      |
| 風（カゼ）                 | 東・南・西・北で表す、席の位置や局の区切りのこと。        |      |
| 么九牌（ヤオチューハイ）   | 1と9の数字牌、または字牌をまとめた呼び方。                |      |
| 中張牌（チュンチャンパイ） | 2から8までの数字が書かれた牌のこと。                      |      |
| 自風牌（ジカゼハイ）       | 自分の座席の風と同じ字牌のこと。                          |      |
| 場風牌（バカゼハイ）       | その局の場の風と同じ字牌のこと。                          |      |
| ドラ                       | 持っているとボーナス点がもらえる特別な牌のこと。          |      |
| 王牌（ワンパイ）           | 山の最後に取っておく、配らない14枚の牌のこと。            |      |

## 手牌・進行に関する用語

| 用語                   | 説明                                                      | 画像 |
| ---------------------- | --------------------------------------------------------- | ---- |
| 手牌（テハイ）         | 自分が持っている牌のこと。ふつう13枚か14枚。              |      |
| 配牌（ハイパイ）       | ゲームの最初に配られる自分の手牌のこと。                  |      |
| 山（ヤマ）             | まだ配られていない、積まれた牌のこと。                    |      |
| ツモ                   | 山から自分で牌を1枚引くこと。                             |      |
| 打牌（ダハイ）         | 自分の手牌から牌を1枚選んで捨てること。                   |      |
| 河（ホー）             | みんなが捨てた牌を並べておく場所のこと。                  |      |
| 捨て牌（ステハイ）     | 自分が捨てた牌のこと。河に並べます。                      |      |
| ポン                   | 他の人が捨てた牌をもらって、同じ牌を3枚そろえること。     |      |
| チー                   | 自分の左隣の人が捨てた牌をもらって、数字を3つ並べること。 |      |
| カン                   | 同じ牌を4枚そろえること。                                 |      |
| 鳴き（ナキ）           | ポン・チー・カンをして、他の人の牌をもらうこと全般。      |      |
| 副露（フーロ）         | ポン・チー・カンをして、牌を表向きに並べること。          |      |
| 面前（メンゼン）       | 1度も副露をしていない状態のこと。                         |      |
| 聴牌（テンパイ）       | あと1枚で和了（アガリ）になれる状態のこと。               |      |
| 和了（ホーラ・アガリ） | 役がそろって、上がることができた状態のこと。              |      |
| ロン                   | 他の人が捨てた牌で和了すること。                          |      |
| 放銃（ホージュウ）     | 他の人がロンする牌を、自分が捨ててしまうこと。            |      |
| 流局（リュウキョク）   | だれも和了しないまま、その局が終わること。                |      |

## 役・点数に関する用語

| 用語               | 説明                                           | 画像 |
| ------------------ | ---------------------------------------------- | ---- |
| 役（ヤク）         | 和了するために必要な、手牌の形のルールのこと。 |      |
| 飜（ハン）         | 役の大きさを表す数のこと。多いほど高得点。     |      |
| 符（フ）           | 点数を計算するときに使う、細かい点数の単位。   |      |
| 順子（シュンツ）   | 同じ種類の数字が3つ連続でそろった形のこと。    |      |
| 刻子（コーツ）     | 同じ牌が3枚そろった形のこと。                  |      |
| 対子（トイツ）     | 同じ牌が2枚そろった形のこと。                  |      |
| 雀頭（ジャントウ） | 手牌の中にある、対子（ペア）の部分のこと。     |      |
| 面子（メンツ）     | 順子・刻子など、手牌を作る3枚1組のまとまり。   |      |

## ゲーム進行に関する用語

| 用語                   | 説明                                             | 画像 |
| ---------------------- | ------------------------------------------------ | ---- |
| 親（オヤ）             | その局で最初に牌を配られ、先に打つ人のこと。     |      |
| 子（コ）               | 親ではない、残り3人のプレイヤーのこと。          |      |
| 他家（ターチャ）       | 自分以外の3人のプレイヤーのこと。                |      |
| 局（キョク）           | 1回のゲームのひとまとまりのこと。                |      |
| 本場（ホンバ）         | 親が続いたときなどに増える、局のカウントのこと。 |      |
| 供託（キョウタク）     | リーチなどで場に置かれたままの点数のこと。       |      |
| 多牌（ターハイ）       | 手牌の枚数が多すぎてしまう反則のこと。           |      |
| 少牌（ショウハイ）     | 手牌の枚数が足りなくなってしまう反則のこと。     |      |
| ちょんぼ               | ルールに反してしまい、罰を受けること。           |      |
| おっかけ               | 他の人のリーチのあとに、自分もリーチすること。   |      |
| 半荘（ハンチャン）     | 東・南の2回まわる、標準的なゲームの単位。        |      |
| 東風戦（トンプウセン） | 東だけで終わる、半荘より短いゲームの単位。       |      |
